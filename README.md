## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/hokarimasahiro/neopixcelclockv2** and import

## Edit this project ![Build status badge](https://github.com/hokarimasahiro/neopixcelclockv2/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://makecode.microbit.org/](https://makecode.microbit.org/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/hokarimasahiro/neopixcelclockv2** and click import
